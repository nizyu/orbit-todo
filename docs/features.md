# Orbit Todo - 機能一覧 (Features)

Orbit Todo は、思いつきのメモから詳細なタスク実行までをシームレスに繋ぐ、Firebaseベースのタスク管理アプリケーションです。
以下に、主な機能とその利用方法をまとめます。

## 📥 1. Inbox (インボックス)

頭の中にあるアイデアやタスクの種、あるいは単なるメモを素早く記録するための場所です。

- **クイックキャプチャ**: テキストボックスから思いついた内容をすぐに保存できます。URLを含めると自動でリンク化されます。
- **フィルタリング**: 「未対処のみ」「Memoのみ」の切り替えや、キーワード検索によって大量のアイテムから必要なものを絞り込めます。
- **AIによる変換**:
  - **タスク化**: メモ内容をAIが解析し、具体的なタイトルと達成に必要なサブタスク（チェックリスト）を自動生成して Tasks に移行します。
  - **アイデア化**: 思いつきをAIが展開し、調査計画（プロジェクトの骨組みや検討事項）として Plans に移行します。

## 📝 2. Plans (プラン)

Inboxから「アイデア化」されたものや、長期的に検討したいプロジェクトの計画を練るための場所です。

- **Markdownエディタ**: リッチなテキストフォーマット（Markdown）で計画や調査内容を記述できます。
- **AIリファイン (洗練)**: 計画の内容に対して「もっと具体的に」「タスクをリストアップして」などの指示（プロンプト）を与えることで、AIが内容をアップデート・改善してくれます。
- **タスクへの変換**: 計画が十分に練られたら、内容から具体的なタスク（Task）を生成して Tasks に移行できます。

## ✅ 3. Tasks (タスク)

具体的な行動（タスク）を管理・実行するためのメイン画面です。

- **2カラム / オーバーレイ レイアウト**: デスクトップでは左側にツリー、右側に詳細画面を表示。モバイル環境では選択時に詳細画面が前面に表示される、直感的でレスポンシブなUIを採用しています。
- **カテゴリ管理**: タスクはフォルダのようにカテゴリ（親・子階層）で分類できます。
- **タスクの詳細管理**:
  - **チェックリスト**: 一つのタスクをさらに小さなステップに分解できます。
  - **期限設定 (Until Date)**: タスクの締め切り日を設定できます。
  - **ステータス管理**: `OPEN` (未着手), `PENDING` (保留・待ち), `COMPLETED` (完了) の3つの状態を切り替えて進捗を管理します。
- **AI サブタスク提案**: タスクの粒度が大きい場合、AIがタスクの内容や所属カテゴリを考慮して、適切なチェックリスト（サブタスク）を自動提案してくれます。

## 🛠 技術スタック

- **フロントエンド**: React (Vite), Tailwind CSS, TanStack Router
- **バックエンド**: Firebase (Authentication, Firestore, Firebase GenAI SDK)
- **AI機能**: Google Gemini (via Firebase Vertex AI)
